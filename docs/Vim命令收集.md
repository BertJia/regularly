## 常用命令收集
1. 更改小写下划线命名为小驼峰命名:
操作名令: %s/_\(.\)/\U\1/gc
2. 增加行注释, 块操作:<C-v> (在Windows下的vim，你需要使用 <C-q>而不是 <C-v> ，<C-v> 是拷贝剪贴板)
操作名令: gg <C-q> j I// ESC
```
快速注释
	1. 按 ESC进入命令行模式；
	2. ctrl+q进入块选择模式；
	3. 操作 k和 j进行上下行选择；
	4. 按大写 I进入插入模式，输入注释符 //或 #(shell脚本注释符)；
	5. 最后按下 ESC即可完成注释。
取消注释
	1. 按 ESC进入命令行模式；
	2. ctrl+q进入块选择模式；
	3. 操作 k和 j进行上下行选择,还可以操作 h和 l键进行左右控制；
	4. 按 d键删除注释符；
	5. 最后按下 ESC即可完成取消注释。
  ```
